# 3.1 源码的下载与使用

本项目遵循`GPL-2.0`开源协议，关于此协议的更详尽的介绍请参考☛[GPL-2.0开源协议][1]。项目中使用了开源数学包`Colt`，关于此包更详尽的介绍请参考☛[Colt科学计算][2]。为了广大研究人员更方便的共享该项目的研究成果，在项目`Library`目录下，提供了`Colt`包的下载，如果您使用的编辑器是IDEA，那么可以按照以下步骤导入该项目和该包。

**第一步**，复制UMAD项目的gitee地址：

> `https://gitee.com/UMAD/UMAD.git`

**第二步**，打开IDEA，选择`Get from Version Control` 从gitee上导入，稍等片刻IDEA自动导入即可。



![导入UMAD][3]

*注意：由于IDEA默认不支持gitee，所以需要手动安装gitee插件，gitee插件的安装本文不会细讲，笔者的电脑是已经配置好gitee插件的。*

**第三步**，选择 `File –> Project Structure ` 打开项目配置文件，然后选择 `Libraries` 导入项目源目录下提供的`Colt` 包即可。

![导入colt][4]

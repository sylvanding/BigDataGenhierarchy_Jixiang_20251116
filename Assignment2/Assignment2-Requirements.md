# Assignment2-Requirements

在Assignment1的基础上实现基本的度量空间相似性查询功能。具体任务如下：

1. 用线性扫描实现范围查询，knn和dknn查询。
2. 实现Pivot Table。
3. 以友好易验证的方式证明上述各项功能的正确性，每项功能至少3个有代表性的测试。
4. 探索、揭示并分析不同支撑点和不同支撑点个数带来的性能差异。

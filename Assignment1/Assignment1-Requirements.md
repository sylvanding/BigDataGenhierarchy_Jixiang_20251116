# Assignment1-Requirements

开发一个基本的度量空间数据处理系统，采用UMAD数据集测试。具体任务如下：

1. 实现度量空间数据抽象父类。
2. 实现`度量空间距离函数`抽象父类，包含任意两个数据元素之间距离计算的通用接口。
3. 作为度量空间数据类型的子类，定义向量数据类型，可以从UMAD数据集（1.2.1 Vector数据集）读入指定纬度和数量的向量。作为度量空间距离函数父类的子类，定义闵可夫斯基距离。
4. 作为度量空间数据类型的子类，定义蛋白质序列类型，可以从UMAD数据集（1.2.5 Protein数据集）读入指定长度和数量的蛋白质序列。作为度量空间距离函数父类的子类，定义基于mPAM的alignment距离。
5. 以友好易验证的方式（如显示计算过程，与容易手工计算的例子对照等）证明代码的正确性，每种数据类型至少3个有代表性的测试。

## Assignment 4 任务概述

### 核心任务

实现和对比分析**三种使用3个pivot的度量空间树状索引**：

1. **3-pivot MVPT（多优势点树）**
   - VP树的多pivot扩展
   - 球形嵌套划分
   - 8个子区域（2³）

2. **3-pivot CGHT（完全广义超平面树）**
   - GH树的多pivot扩展
   - 基于距离差的超平面划分
   - 4或8个子区域

3. **3-pivot 完全线性划分树**
   - 在支撑点空间中进行线性划分
   - 正交划分策略
   - 8个子区域（2³）

### 分值分配

| 任务 | 分值 |
|------|------|
| 实现三种索引（批建+范围查询） | 30分 |
| 正确性验证（2个数据集，树高≥2） | 10分 |
| 理论对比分析 | 30分 |
| 实验对比分析 | 30分 |

### 关键要求

- ✅ 三种索引使用**相同的支撑点选择算法**
- ✅ 展示批建和范围查询的**算法及代码**
- ✅ 在至少**2个数据集**上验证
- ✅ 树高至少**2层**
- ✅ 使用**LaTeX**编写实验报告
